# newNed
Checks to see if a new Nedroid comic has been uploaded to the homepage of the Nedroid website


1. Program starts with a predefined date to search for
2. Check string to see if this predefined date is missing
3. If it is missing, this means there is a new comic

4. Because there is a new comic, the aformentioned predefined date is no longer relevant
5. A new date variable needs to be checked and can reused/updated
6. Keep checking every x amount of time
